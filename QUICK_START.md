# Quick Start Guide - New Features

## Getting Started with Admin & Teacher Features

### For Admins:

1. **Login as Admin**:
   - Open the app
   - Click "Admin" radio button
   - Enter admin email and password
   - Click "Login"

2. **Manage Users**:
   - Click "+ Add User" button
   - Enter email, name, password, select role
   - Click "Create User"
   - To delete: Click "Delete" button next to user

3. **System Management**:
   - Click any button in "System Management" section
   - Features expanding soon!

---

### For Teachers:

1. **Login as Teacher**:
   - Open the app
   - Click "Teacher/Academy" radio button
   - Enter teacher email and password
   - Click "Login"

2. **Create a New Test**:
   - Click "+ Create Test" button (green)
   - Enter test name, select module and difficulty
   - Click "+ Add Question"

3. **Add Questions**:
   - Enter question text (you can use LaTeX: $\frac{1}{2}$)
   - Enter all 4 multiple choice options
   - Select which option is correct
   - Add explanation (why this answer is right)
   - (Optional) Upload question image
   - Click "Save Question"

4. **Manage Questions**:
   - Click "Edit" to modify a question
   - Click "Delete" to remove a question

5. **Save Your Test**:
   - Click "💾 Save Test" button (floating at bottom right)
   - Test is saved! ✓
   - Return to dashboard

6. **View Student Progress**:
   - See all student test results in the table
   - Send reports via WhatsApp (if parent phone available)
   - Review individual student attempts

## Tips & Tricks

1. **Question with Math**: Use LaTeX syntax like `$\sqrt{2}$` or `$\frac{a}{b}$`

2. **Question with Image**: Upload images for visual problems (diagrams, charts, etc.)

3. **Multiple Choice**: Make sure to select which option is correct!

4. **Explanation**: Write clear explanations - students learn from these!

5. **Test Types**: 
   - Module 1: General SAT questions
   - Module 2 Easy: Easier math problems
   - Module 2 Hard: Challenging math problems
